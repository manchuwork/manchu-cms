manchuweb 满族空间
满族学习网站 ，采用满文竖版的方式，阅读方式为从上到下，从左到右 推荐用最新的chrome浏览器浏览，目前暂时主要支持PC版，移动版未做适配

基于laravel5.4

目前提供的功能
字典 划词，基于字典 短句 图书目录

网站地址
www.manchu.work 满族空间
## 环境需求

* Composer
* PHP >= 7.1.3
* OpenSSL PHP 扩展
* PDO PHP 扩展
* Mbstring PHP 扩展
* Tokenizer PHP 扩展
* XML PHP 扩展
* Ctype PHP 扩展
* JSON PHP 扩展
* Mysql 5.7+
> 中文分词会占用内存，请将 `php.ini` 的 `memory_limit` 参数调整至 256M


注：要先配置好数据库，默认用户: 
admin@manchu.work/admin
